ROS Basics Project

  • Tech Stack: Python, MoveIt, FlexBE, OMPL, ROS
  • Github URL: Project Link

Video Demonstration of the Project Demo1 and Demo2.
The outcome of the project was as follows

  • Using a turtlebot to plan a dynamic path for the AGV to follow from one part of the factory to the other.
  • Using MoveIt and OMPL to plan the motion of the robotic Arm.
  • Using the FlexBE Application for controlling the behaviour of the automated Factory.
    • This Project is used for clarifying ROS concepts. These concepts are listed below

    • Publisher & Subscriber
    • Action Server & Client
    • Topic & Message
    • Launch Files
    • Parameters Server
    • Rosbag files
    • URDF and XML files for models
    • Finite State Machines
